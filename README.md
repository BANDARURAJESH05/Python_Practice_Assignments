# Python_Practice_Assignments

## Overview

This repository contains Python practice programs that I solved while learning Python programming. It includes assignment-based problems that helped me improve my coding skills and logical thinking. These coding questions cover commonly asked problems for interviews, campus placements, and technical assessments.

---

## Topics Covered

This repository includes practice programs on:

* **Python Lists**

  * List operations
  * Finding maximum/minimum
  * Counting elements
  * Traversing lists using loops

* **Python Dictionaries**

  * Accessing key-value pairs
  * Adding and updating values
  * Checking keys
  * Dictionary operations using loops

* **Python Strings**

  * String reversal
  * Palindrome checking
  * Character counting
  * String operations using loops

* **Loop Practice**

  * `for` loop
  * `while` loop
  * Logic building without using shortcuts

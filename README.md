## Repository Content

This repository mainly contains assignment-based Python programs on different concepts.

### Python List Programs

This section contains programs related to Python Lists.

Programs practiced include:

* Count number of elements in a list without using `len()`
* Find maximum and minimum value in a list
* Remove first occurrence of an element
* Find average of list elements
* Practice list traversal using loops
* Basic list operations and problem solving

---

### Python Dictionary Programs

This section contains programs based on Dictionary concepts.

Programs practiced include:

* Access value using key
* Add new key-value pair in dictionary
* Check whether a key exists in dictionary
* Find sum of dictionary values
* Get all keys from dictionary
* Dictionary traversal using loops
* Practice dictionary operations and logic building

---

### Python String Programs

This section contains programs related to String handling and operations.

Programs practiced include:

* Count number of characters without using `len()`
* Reverse a string without using slicing `[::-1]`
* Check palindrome without using shortcut methods
* Find most common character in a string
* Check whether two strings are anagrams
* String manipulation using loops

---

Concepts used:
